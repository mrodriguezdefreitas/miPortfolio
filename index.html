Que servicios ofrezco?

Estructura

A través de HTML creo y doy formato a las páginas web. Genero y organizo el contenido de una página. Utilizo encabezados, párrafos, listas, enlaces y tablas. También incorporo elementos multimedia, como videos e imágenes.

Apariencia

Mediante CSS doy estilo y diseño a las páginas web. Defino colores, fuentes, tamaños de texto, márgenes, bordes y otros estilos visuales. Además, realizo diseños que se adaptan a diferentes dispositivos y tamaños de pantalla.

Diseño

Poseo conocimientos básicos de Photoshop, lo cuál me permite desenvolverme de una manera óptima a la hora de editar imágenes o diseñar logotipos.
